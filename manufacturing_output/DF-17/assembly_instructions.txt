DF-17 ASSEMBLY INSTRUCTIONS
========================================

1. COMPONENT VERIFICATION
   - Verify all components per BOM
   - Check material certifications
   - Confirm dimensional tolerances

2. ASSEMBLY SEQUENCE
   1. Inspect all components per BOM
      Tool: Calipers, CMM

   2. Mount propulsion section to assembly fixture
      Torque: 120 Nm
      Tool: Torque wrench

   3. Install guidance section with alignment pins
      Torque: 80 Nm
      Tool: Torque wrench

   4. Install warhead section with explosive safety interlocks
      Torque: 150 Nm
      Tool: Torque wrench

   5. Mount fin assemblies with aerodynamic alignment
      Torque: 60 Nm
      Tool: Torque wrench

   6. Install avionics and wiring harness
      Torque: 20 Nm
      Tool: Torque screwdriver

   7. Pressure test propulsion system
      Tool: Pressure test rig

   8. Final inspection and acceptance testing
      Tool: Test equipment

3. QUALITY CHECKS
   - Pressure test: 5000 psi for 30 min
   - Leak check: < 1% pressure drop
   - Alignment check: < 0.1mm tolerance
   - Electrical continuity: All systems
